Project Title:

MediAlert: Medicine Reminder Website
Abstract:
MediAlert is a web-based medicine reminder system designed to help users manage their medication schedules effectively. The system allows users to add medicines with dosage details, timings, and duration, and sends timely reminders to ensure medicines are taken on schedule. It aims to improve medication adherence, reduce missed doses, and support better health management. With a simple and user-friendly interface, MediAlert provides secure data handling and reliable reminders, making it useful for patients, elderly users, and caregivers.

Domain and Technology:
Domain: Web-Based Application Development, Healthcare Automation
Application Area: Medicine Reminder & Health Management

Backend: Node js
